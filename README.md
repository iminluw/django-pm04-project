## Веб-приложение "Менеджер задач" на Django

**Студент:** [Твоё Имя]
**Группа:** [Твоя Группа]

### Описание:
Универсальное веб-приложение для управления задачами, реализованное в рамках дифференцированного зачета.

### Функционал:
*   Управление задачами через панель администратора Django.
*   Отображение актуального списка задач на главной странице.
*   Статусы выполнения задач (В работе / Завершено).
*   Использование Bootstrap для оформления интерфейса.

### Как запустить:
1. Установить зависимости: `pip install django`
2. Выполнить миграции: `python manage.py migrate`
3. Запустить сервер: `python manage.py runserver`
